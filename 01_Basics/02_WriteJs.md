# How to Write JavaScript

We can write JS Code in any Code Editor or Notepad


# Ways to write
1. Inline JS 
    - directly writing JS code to the element
    - ex: <buttononclick="alert('Hello')">Click Me</button>

2. Internal JS
    - inside the html file but separately in <script> tag
    - <script> console.log("hello")</script>

3. External JS
    - Writing in a separate .js file and then including it later
    - <script src="script.js"></script>
