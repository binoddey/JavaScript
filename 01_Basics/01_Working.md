# How Websites Work with JavaScript

## The Big Picture
At a high level, a website runs on three core technologies:
- **HTML** → Structure
- **CSS** → Styling
- **JavaScript** → Logic + Interactivity

When you open a website:
1. Browser sends request to server
2. Server sends back HTML, CSS, JS
3. Browser builds the page
4. JavaScript makes it interactive


## What is JavaScript 
JavaScript is:
- A high-level programming language
- Dynamically typed
- Single-threaded
- Used for web development (frontend + backend)

Runs in:
- Browser
- Node.js (server-side)
